## Slot-lettering verification (R-M19-6, empirically confirmed, not assumed)

openMSX's general CLI docs (`-cart`, `-carta`..`-cartd`) do not by themselves say which lettered slot lands in THIS machine's primary slot 1 vs. 2. Verified live via a WSL Tcl probe BEFORE any slot-specific claim below: two distinguishable synthetic `Mirrored` images (marker bytes 0xAA / 0xBB) were mounted via `-carta`/`-cartb`; `debug write ioports 0xA8 0xF7` (routing CPU page 1 to primary slot 1, this project's own SlotBus bit layout) then `debug read memory 0x4000` returned `0xAA` (the `-carta` file's marker); `debug write ioports 0xA8 0xFB` (page 1 -> primary slot 2) then the same read returned `0xBB` (the `-cartb` file's marker). **Conclusion: `-carta` = primary slot 1, `-cartb` = primary slot 2, for this machine, on this installed openMSX 19.1.** This harness therefore uses `-carta` for its primary-slot-1 claim below.

## Result

Mechanism: this emulator's `--parity-trace` (extended M19-S5 to recognize `--cart1`/`--cart1-type` via the SAME parser/loader as the default run path) vs. openMSX driven by a chained single-step Tcl script that mounts the IDENTICAL cartridge file via `-carta -romtype 8kB` and injects the IDENTICAL driver-program bytes into RAM before single-stepping -- diffed per-instruction (PC, opcode, every register/flag, including AF where the read-back marker bytes land) via `tools/trace-diff.py`. Because both sides load an IDENTICALLY-AUTHORED synthetic cartridge, the expected AF values at the three `LD A,(addr)` steps (0x00 reset-default bank0, then 0x03 after the bank-switch write, then 0x01 for the unaffected window-slot) are fully known in advance -- this is a genuine CONTENT-bearing comparison, not merely architectural.

- Emulator instructions (A): 8
- openMSX instructions (B): 8
- **Result: ARCHITECTURAL + CONTENT PARITY -- EMPTY DIFF** over all 8 instructions (including the read-back marker bytes in AF).

Raw traces: `build\m19_cartridge_trace_A.txt` (emulator), `build\m19_cartridge_trace_B.txt` (openMSX). Diff: `build\m19_cartridge_trace_diff.md`.
